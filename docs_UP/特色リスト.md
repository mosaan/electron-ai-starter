# 特色リスト（Feature List）

> 要求候補のリスト。作業の立案に使われます。

本文書は、Electron AI Starter Templateプロジェクトにおける主要な特色（Feature）を列挙したものです。これらは既存の実装と設計文書から抽出された要求候補であり、今後の反復における作業の優先度づけと立案の基礎となります。
